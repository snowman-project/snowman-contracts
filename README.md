# Snowman Project

## How to Install

```sh
yarn
```

## How to Develop and Test Locally

### Compile

```sh
yarn run compile
```

### Start a local server forked from mainnet

```sh
yarn run mock
```

### Deploy to local

```sh
yarn run deploy
```

### Test from local

```sh
yarn test
```

> You don't need to compile or deploy before testing.
